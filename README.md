# notepadsh
An easy to use notepad that I'm making to practice bash scripting


<h2>Installation</h2>

```
git clone https://github.com/SkylerHope/notepadsh.git

cd notepadsh

chmod +x install.sh

./install.sh
```

<h2>Usage</h2>

```
sh /notepadsh/notepad.sh
```
OR
```
cd notepadsh

./notepad.sh
```

<h2>Clean your notes</h2>

```
cd notepadsh

./clean.sh
```
OR
```
sh /notepadsh/clean.sh
```

<h2>Uninstall</h2>

```
cd notepadsh

./uninstall.sh

cd ..

rm -rf notepadsh
```
